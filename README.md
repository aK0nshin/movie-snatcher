# movie-snatcher



## Quick start

Токен для апи кинопоиска можно получить в https://t.me/kinopoiskdev_bot

```
cd movie-snatcher
cp .env.example .env
# Вписать в .env свои настройки (KINOPOISK_TOKEN как минимум)
docker compose up --build --wait
```
